Quanto aos links

Os links devem indicar a página que vão pelo seu nome, evite usar textos como "clique aqui" no lugar disso use "*página que queremos ir*".
Por exemplo para ir para o site da wikipedia use esse link.

💀 Quanto às listas 💀

Para criar listas usamos ul para listas que não precisam de uma ordem e ol para listas ordenadas, por exemplo:

  • Uma receita
  • Uma lista de tarefas
  • Uma lista de filmes
  • Uma lista de links
  • e mais...

E com ordem

  1. Uma lista de melhores jogadores
  2. Um ranking das melhores personagens femininas de jogos nintendo
  3. Uma classificação das melhores lojas varejistas da cidade

  5. etc...
